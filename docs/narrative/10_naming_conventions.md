# Naming Conventions

## Status

**Stub.** Reserved for the rules that keep names in *Purity Wars* feeling like they come from one world. The world-level vocabulary is established in `02_world_setting.md`; this doc is where names get governed.

## Established Names (As Of Now)

- **Ulcantron** — the world.
- **Bragaldita** — the leftmost of the two main continents.
- **Zortinus** — the rightmost of the two main continents.
- **Azzentia** — the kingdom in which *Purity Wars* takes place, in the bottom-right of Bragaldita.
- **Magik** — magic, as spelled in-world, with a *k*.
- **The Rite of Naming** — the Azzentian coming-of-age class ritual.

## What This Doc Will Contain When Filled In

- **Rules for new place names.** Phonetic patterns acceptable inside Azzentia (drawn from the existing names above), and patterns to avoid (anything that reads as joke-fantasy or pulled from a contemporary language too cleanly).
- **Rules for character names.** Pure-faction families, Mixed-faction families, regional variants, civic-rank suffixes if any.
- **Rules for institutional names.** Temples, knight orders, academies, trade guilds.
- **Class names and subclass names.** Capitalisation, pluralisation, abbreviations.
- **Names that are off-limits.** Anything that overlaps with real religions, real political movements, or real historical regimes in a way that would short-circuit the allegory.
- **A short approved-names ledger** — the canonical spelling of every named character, region, institution, and notable artefact, so they don't drift across docs and dialogue files.

## Authoring Rules When Filled In

- Once a name lands here it is canonical. Don't change it casually; renaming after content is written breaks save data, dialogue files, and player familiarity.
- New names should be sanity-checked against the existing established names for phonetic compatibility.
- If a name has more than one in-world spelling (formal vs. colloquial), record both and mark which is preferred in narration.
